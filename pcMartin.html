La programmation de jeux sur PC

À la différence du développement sur consoles, les configurations pour deux joueurs donnés ne sont pas les mêmes. Il faut donc prévoir différents réglages graphiques permettant d’exploiter les possibilités offertes entre différents PC, qu’il soit plus puissant ou plus modeste en terme de configuration. Le prix de développement sur PC est plus faible car il n’est pas nécessaire de posséder une license pour vendre son jeu contrairement aux consoles et aux mobiles. Il existe une accessibilité différente en fonction des OS utilisés, on aura donc beaucoup plus de jeux présents sur l’OS windows que sur son concurrent Mac, ou encore linux. La programmation de jeu se fait beaucoup par l’intermédiaire de moteurs de jeux tels que Unity ou Unreal Engine, qui permettent la programmation d’un jeu via cette plateforme qui va structurer sa programmation en l’accompagnant grâce à une interface.

Les jeux en java

Les jeux flashs représentent une grande partie des jeux vidéos, car leur positionnement sur internet de part l'hébergement de site leur permet d'être diffuser à grande échelle. Dans cette optique de format apparait plusieurs langages de programmation notamment le java. Le java est un des langages les plus utilisés, il est utilisé pour créer des applications d'entreprises, mais est également très pratique dans la programmation orientée objet. Il est avantageux de part sa portabilité, c'est-à-dire qu'il est executable sur tous systèmes et est rapide à l'execution. Il comporte cependant des problèmes, en effet ce n'est pas le langage le plus beau et lisible et le temps de chargent de la JVM (Java Virtual Machine) peut être long.

Le logo de Java avec la tasse est très connu et représente la marque de fabrique de java.

On retrouve de nombreux site en ligne hébergeant des jeux vidéos comme ici par exemple le site miniclip.

Les jeux en flash

On retrouve également pour la programmation web le module flash d'extension .swf. Il représente des données intéractive entre les pages web dans un format fermé de données d'animations. Ce plugin adobe flash n'est pas seulement nécessaire pour les jeux vidéo, mais on le retrouve aussi beaucoup dans l'animation vectorielle de sites. Les jeux flashs sont souvent liés aux jeux en HTML5 et en JS, deux langages web avec utilisation de la programmation orientée objet auniveau du JS qui est l'un des langages les plus utilisés. On a à gauche, le très célèbre module adobe flash player utilisé sur de nombreux ordinateurs, servant à afficher des animations vectorielles dans des espaces fermés..

Les jeux en code natif

Il existe de nombreux jeux sous PC dit en code natif. C'est à dire qu'ils sont reconnus directement par le processeur et n'ont pas besoin d'interpréter pour être exécutés.
